# Research Plan: Deep Research Skill for Claude Code

## Objective
Find the best available deep research skill/plugin for Claude Code CLI — something that replicates or exceeds the "Deep Research" mode available in Claude.ai's browser interface. Prioritize options that use sub-agents effectively for speed and accuracy.

## Research Perspectives

### 1. Official Anthropic Deep Research Features
- What is Claude.ai's "Deep Research" mode and how does it work?
- Has Anthropic released any official deep research tooling for Claude Code?
- Are there any announcements about bringing deep research to Claude Code?
- What does the Claude Code documentation say about research capabilities?

### 2. Community Claude Code Skills & Plugins (GitHub)
- Search GitHub for "claude code deep research skill" and variants
- Look for popular repos that add research capabilities to Claude Code
- Check the Claude Code plugins ecosystem for research-focused plugins
- Look for skills that use multi-agent/sub-agent patterns for research

### 3. Claude Code Sub-Agent Research Patterns
- How does Claude Code's Agent tool enable deep research workflows?
- What sub-agent architectures exist for parallel research?
- How do existing skills (like research-and-implement) compare?
- What patterns do power users employ for research tasks?

### 4. Third-Party AI Research Tools & Agents
- What standalone AI research agents exist (not Claude Code specific)?
- Are any of these adaptable as Claude Code skills/MCP servers?
- How do tools like Perplexity, Tavily, Exa compare?
- Are there MCP servers designed for deep research?

### 5. Community Discussion & Recommendations
- What do Reddit, HN, Discord, and forum discussions say about deep research in Claude Code?
- What setups do power users recommend?
- Are there blog posts or tutorials on building research workflows?
- What are the known limitations and workarounds?

### 6. Comparison: Claude.ai Deep Research vs Claude Code Research
- Feature comparison: what can browser deep research do that CLI can't?
- What advantages does Claude Code have (file persistence, sub-agents, automation)?
- Is there a way to access Claude.ai's deep research API programmatically?
- Gap analysis: what's missing and what fills the gaps?

## Success Criteria
- Identify at least 3 concrete options (official, community, or DIY)
- Rank them by: quality, ease of installation, sub-agent usage, accuracy guarantees
- Provide clear installation/setup instructions for the top recommendation
- Honest assessment of what works vs what's vaporware
